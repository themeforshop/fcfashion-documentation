    Home Page Sections

    Below guide how to use section of Freshcom Fashion #1

    Slideshow

    • If the section existed (used preset), you only need click to section to edit.

    • If the store don't have it, only need click to button and select Slideshow.

    General Settings: Display on, Show Full width, Style value (Margin,Z-Index), Slide effect (include 4 effects: Sliding, Fading, Cube, Flipping ), Auto-rotate slides, Slide width in px (Desktop), Slide height in px (Desktop), Enable Navigaiton, Navigation Position and Enable Pagination.

    The Important area is the Content area (bottom). It will listing all slide of Slideshow. Click to Slide to edit. If you want add more slide, click to

    For the Image slide, You will got options:

    • Slide Background: Upload image to show. Also can set Slide link, Captions Entrances Effect and Captions Exits Effect

    • Title: It's heading of the caption.

    • Description: it's description of the caption.

    • Action: it't action button of the caption.

    • Other captions (maximun is 6) you can set below.

    With each caption element, you need set the style and position to make it show the like your idea. These options area Style, horizontal position(x), align with horizontal position, vertical position(y), layer order(z)


    For the Video slide, You will got options:

    • Video link: input the embed video.

    • Auto play.

    • Image loader.

    Banner Custom

    • If the section existed (used preset), you only need click to Banner Custom section to edit.

    • If the store don't have it, only need click to button and select Banner Custom.

    There are Left and Right banners options: each option will have options to upload image, input caption



    Featured Products

    • If the section existed (used preset), you only need click to Featured Products section to edit. The name will depend on Title text. So, you need temporary remove the title to have the right section name.

    • If the store don't have it, only need click to button and select Products Slider.

    General Settings: Title, Collection to show products, Slider and Responsive settings

    Responsive settings.


    Collections

    • If the section existed (used preset), you only need click to Collections section to edit.

    • If the store don't have it, only need click to button and select Collections.

    Content area: listing all collection element. Each element will have options to upload image, caption and Responsive settings.



    New Arrivals

    • If the section existed (used preset), you only need click to New Arrivals section to edit. The name will depend on Title text. So, you need temporary remove the title to have the right section name.

    • If the store don't have it, only need click to button and select Products Banner Slider.

    General Settings: Display on, Title, Collection to show products and Responsive width.


    Deal of the Day

    • If the section existed (used preset), you only need click to Deal of the Day section to edit. The name will depend on Title text. So, you need temporary remove the title to have the right section name.

    • If the store don't have it, only need click to button and select Banner with Countdown.

    General Settings: Display on, Title, Sub-title, Timer, Action options and Background image.


    Products Column

    • If the section existed (used preset), you only need click to Products Column section to edit. The name will depend on Title text. So, you need temporary remove the title to have the right section name.

    • If the store don't have it, only need click to button and select Products Column.

    General Settings: Display on, Tlimit product to show.

    Content area: listing all columns. Each column have option to upload title, select collection to show products.


    Information

    • If the section existed (used preset), you only need click to Information section to edit. The name will depend on Title text. So, you need temporary remove the title to have the right section name.

    • If the store don't have it, only need click to button and select Information.

    General Settings: Display on.

    Content area: will listing all information element.
